# 🚀 1단계 - 문자열 계산기

## 기능 목록
* 입력
  - [X] 입력 문자열의 숫자와 연산자 사이에는 반드시 빈 공백 문자열이 있어야 한다.
  - [X] 입력 값이 null이거나 빈 공백 문자일 경우 IllegalArgumentException를 반환한다.
  - [X] 사칙연산 기호가 아닌 경우 IllegalArgumentException를 반환한다.
* 계산기
  - [X] 더하기
  - [X] 빼기
  - [X] 곱하기
  - [X] 나누기
    - [X] 나눗셈의 경우 결과 값을 정수로 반환해야 한다.
* 출력
  - [X] 계산 결과를 출력한다.
